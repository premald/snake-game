# Snake Game

Simple browser-based Snake game.

## Play Locally

1. **Install dependencies**
   ```bash
   npm install
   ```
2. **Run the game**
   ```bash
   npm start
   ```
3. Open your browser at the local URL shown in the terminal (commonly `http://localhost:3000`).

## Controls

- **Arrow keys / WASD**: Move snake
- **On-screen buttons**: Move snake (mobile/desktop)
- **Swipe on game grid (touch devices)**: Change direction
- **Space**: Pause / Resume
- **R**: Restart game

> Swipe and other direction inputs do not allow an immediate reverse direction.
