# Snake Game

Simple browser-based Snake game.

## Play Locally

1. **Install dependencies**
   ```bash
   npm install
   ```
2. **Run the game**
   ```bash
   npm start
   ```
3. Open your browser at the local URL shown in the terminal (commonly `http://localhost:3000`).

## Controls

- **Arrow keys**: Move snake
- **Space**: Pause / Resume
- **R**: Restart game
