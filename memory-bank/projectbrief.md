# Project Brief

## Project
Snake Game (browser-based)

## Goal
Maintain a simple, responsive Snake game and extend it with small, user-facing features using the agent workflow.

## Key Features
- Core Snake gameplay with keyboard and on-screen controls
- Score and high score tracking
- Player name prompt stored per session

## Constraints
- Vanilla HTML/CSS/JS
- No build step required for the game
- Lightweight UX; avoid heavy dependencies

## Success Criteria
- Feature additions are small, reliable, and easy to understand
- Agent workflow can be reused for future changes
